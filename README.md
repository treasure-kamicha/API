## DESCRIPTION

 I built a React application using create-react-app to practice API consumption and state management. I created a UserList component that uses Axios inside a useEffect hook to fetch user data from the JSONPlaceholder API. The data is stored in a useState hook and rendered dynamically with the map function to display a list of users on the screen. Finally, I applied custom CSS styling to make the interface visually appealing.
